Aplicação web desenvolvida em Laravel consumindo API https://www.balldontlie.io para visualização de dados dos jogadores.

Funcionalidades:
• Listagem de jogadores
• Destaque visual do jogador selecionado
• Responsividade

Tecnologia usada:
•BootsTrap
•JavaScript
•Laravel 12.20.0
•Guzzle
